# Kagi-Kashi（笑）

アプリ制作プロジェクトのやつ

- 縦並びに表示。
- 開錠。施錠。一旦施錠している場合、メッセージを残せる。
- 部室を持っているサークルを上から表示させる。
- お気に入り機能。
- 通知
- 名前検索。
- 活動報告書。
- 引継ぎ設定。

## 手順

1. ログイン
2. 部活動一覧が出ている。（一時施錠：赤、施錠中：灰色、活動中：緑）
3. 部活動をクリック
4. 部活動詳細ページに飛ぶ
5. 活動報告書、状況、鍵の貸し出しが選択できる画面がある。

## 鍵を借りる場合

- その部活に学籍番号が登録されている。
- 学籍番号、名前、鍵番号を入力後借りれる。

## 鍵を返す場合

- ログイン後の詳細ページで返却を選択し色が変われば返却済みになる。

## 環境構築方法

- ctrlキー+Jでターミナルを起動
- 以下のコマンドをターミナルで実行

```bash
1. py -3 -m venv .venv
2. .venv\Scripts\activate
3. pip install -r requirements.txt
```

## 起動方法
- ディレクトリ直下にあるapp.pyを起動

## ディレクトリ構成
kagi-kashi/
├── .venv/           - ライブラリがたくさん入ったフォルダ
├── flaskr/          - メインのフォルダ
│   ├── __pycache__
│   ├── templates/   - htmlファイル入れ
│   └── __init__.py  - 起動の中核となる処理が書かれたファイル。app.pyはこれを呼び出している
├── tests/           - test用のファイル入れ
├── app.py           - これを実行するとアプリが起動する
├── README.md
└── requirements.txt - モジュールが書かれたファイル。環境構築の3行目はこれを参照している
